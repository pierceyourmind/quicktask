# Requirements: QuickTask v1.0

**Defined:** 2026-02-17
**Core Value:** Zero-friction task capture — the moment a task enters your mind, one hotkey and a few keystrokes saves it before it vanishes.

## v1 Requirements (All Implemented)

### App Shell

- [x] **SHELL-01**: Menu bar icon visible in macOS status bar at all times
- [x] **SHELL-02**: App runs as menu bar agent (no Dock icon, LSUIElement)
- [x] **SHELL-03**: Floating panel appears centered on screen (Spotlight-style) when activated
- [x] **SHELL-04**: Panel dismisses on Escape key press
- [x] **SHELL-05**: Panel dismisses on click outside the panel

### Hotkey

- [x] **HKEY-01**: Global keyboard shortcut toggles floating panel open/closed
- [x] **HKEY-02**: Hotkey works when app is in background (any app focused)
- [x] **HKEY-03**: Panel appears in < 200ms perceived latency on hotkey press
- [x] **HKEY-04**: Focus returns to previous app when panel is dismissed

### Task Capture

- [x] **CAPT-01**: Text field is auto-focused when panel opens
- [x] **CAPT-02**: User types task text and presses Return to add task
- [x] **CAPT-03**: Text field clears after task is added, ready for next entry
- [x] **CAPT-04**: Tasks appear in a checklist below the text field

### Task Management

- [x] **TASK-01**: Each task has a checkbox to mark it complete
- [x] **TASK-02**: Completed tasks show strikethrough and reduced opacity (faded)
- [x] **TASK-03**: Completed tasks remain visible in the list (not auto-deleted)
- [x] **TASK-04**: User can delete individual tasks

### Persistence

- [x] **PERS-01**: Tasks persist across app quit and relaunch
- [x] **PERS-02**: Tasks persist across system reboot
- [x] **PERS-03**: Data stored locally as JSON in ~/Library/Application Support/

### Settings

- [x] **SETT-01**: Launch at login toggle (opt-in, defaults to off)
- [x] **SETT-02**: Settings accessible from menu bar icon context menu

## v2 Requirements (Deferred)

- **ENHC-01**: Task count badge on menu bar icon (active task count)
- **ENHC-02**: Drag-to-reorder tasks with drag handles
- **ENHC-03**: Configurable hotkey via recorder UI
- **ENHC-04**: Bulk-clear completed tasks ("Clear all done" button)
- **ENHC-05**: Optional due date field (single date, no recurrence)
- **ENHC-06**: Keyboard shortcut to mark selected task complete
- **ENHC-07**: Plain text export / copy task list to clipboard
